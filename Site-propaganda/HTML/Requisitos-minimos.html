REQUISITOS MÍNIMOS DE HARDWARE POR EXPERIÊNCIA:

Mínima:
Sistema operacional necessário Windows 7 ou superior. Linux Distro de 2014 ou mais recente. MacOS 10.9 ou superior
Memória RAM necessária 4GB
Armazenamento 1GB
Placa gráfica Intel HD Graphics 4000 ou AMD Radeon R5
Processador Intel Core i3-3210 ou AMD A8-7600

Recomendada:
Sistema operacional necessário Windows 10, macOS 10.12 ou uma distribuição Linux recente
Memória RAM necessária 8GB
Armazenamento 4GB (em ssd)
Placa gráfica GeForce série 700 ou AMD Radeon série 200 com OpenGL 4.5
Processador Intel Core i5-4690 ou AMD A10-7800, ou equivalentes
